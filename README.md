# mindnum

Get your mind number.

</div>

## About

A command to get the mind number.
Mind number is the number obtained by breaking down all dates of birth in the Western calendar and adding them to a single digit.

## Usage

```
Usage of mindnum:
  -v, --version
    show version
  -h, --help
    help for mindnum
  get -b, --birthday yyyymmdd
    get your mind number
```
